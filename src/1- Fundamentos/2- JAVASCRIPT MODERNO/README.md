# Modern JavaScript Module
This is the Modern JavaScript module, in which the moderns concepts (ES6) of JavaScript were passed

## Example of Modern Structure in JavaScript
- Default Structure:
```js
    function sum(a, b) {
        return a + b
    }
    console.log(sum(1,2)) // => 3
```
- Modern Structure:
```js
    const sum = () => a + b
    console.log(sum(1,2)) // => 3
```
